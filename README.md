# Bot Turnos Dr. Rossi

**Demo en vivo:** [bot-turnos-dr-rossi.vercel.app](https://bot-turnos-dr-rossi.vercel.app)

Sitio web de gestión de turnos para consultorio médico. Desarrollado con Astro.

## Stack

- **Astro** (generación estática)
- **Tailwind CSS**
- **pnpm**

## Correr localmente

```bash
pnpm install
pnpm dev
```

El sitio estará disponible en [http://localhost:4321](http://localhost:4321).

## Scripts

| Comando | Descripción |
|---|---|
| `pnpm dev` | Servidor de desarrollo |
| `pnpm build` | Build de producción |
| `pnpm preview` | Preview del build local |
